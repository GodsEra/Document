> **导航: 供应商列表**

> 接口地址 /supplier_user/supplierUserList 请求方式 POST

> **传递参数 Request Data : **
```
formdata 格式 或 json 客户端自行选择

string      token                   【口令】        
int/null    limit                   【分页数(默认后台配置)】        
int/null    page                    【页数(默认1)】
```
