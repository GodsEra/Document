> **API 接口返回说明文档**

> **导航: 用户信息**

> 接口地址 /player/team_condition 请求方式 POST

> **传递参数 Request Data : **
```
只有代理 或 总经理 或 商务部  有权限可以访问此接口
formdata 格式 或 json 客户端自行选择

token=xxx 必填
```
