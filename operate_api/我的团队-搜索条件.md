> **API 接口返回说明文档**

> **导航: 用户信息**

> 接口地址 /team/search_condition 请求方式 POST

> **传递参数 Request Data : **
```
只有代理 或 总经理 或 部长 权限可以访问此接口
formdata 格式 或 json 客户端自行选择

token=xxx 必填
b_id=xxx  代理 总经理 角色 需要传此值 部长角色不需要传此值
```
