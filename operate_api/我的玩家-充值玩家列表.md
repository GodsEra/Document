> **API 接口返回说明文档**

> **导航: 用户信息**

> 接口地址 /player/recharge_list 请求方式 POST

> **传递参数 Request Data : **
```
formdata 格式 或 json 客户端自行选择

在线玩家也是这个接口

token=xxx 必填
page=xxx 分页

今日 time_type = today
本周 time_type = week
本月 time_type = month
充值玩家 time_type = already
未充值玩家 time_type = not

不填 DR玩家
```
