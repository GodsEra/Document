> **API 接口返回说明文档**

> **导航: 用户信息**

> 接口地址 logistics/index 请求方式 POST

> **传递参数 Request Data : **
```
formdata 格式 或 json 客户端自行选择

token=xxx 必填
page=xxx 分页码
agent_id=xxx 按区域搜索 区id 不填写 就是查询全部
```
